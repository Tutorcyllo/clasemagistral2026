# ¿Podemos seguir utilizando la clase magistral como herramienta educativa?

**Cuaderno 2 · EduMed UVa · 2026**  
Unidad de Educación Médica · Facultad de Medicina · Universidad de Valladolid

---

## Descripción

Aplicación web interactiva basada en el Cuaderno 2 de la serie *Cuadernos de Información en Educación Médica* de EduMed UVa, titulado originalmente *"Did active teaching kill the lecture star?"*

La app permite a docentes de medicina explorar de forma visual e interactiva:

- El debate entre clase magistral y metodologías activas, y por qué la evidencia es menos concluyente de lo que se ha asumido.
- Las limitaciones inherentes de la clase magistral y cuándo sigue teniendo valor educativo.
- El modelo de las **5 Aes** — un marco práctico para transformar la clase magistral en una experiencia Activa, Ágil, Aplicable, Auténtica y Ampliable — con ejemplos concretos para el aula de medicina.
- Los conceptos clave del documento en formato de referencia rápida.

## Uso

La app es un único archivo HTML autocontenido. Para usarla:

1. Clona el repositorio o descarga `index.html`
2. Ábrelo en cualquier navegador moderno
3. No requiere servidor ni instalación
